# Accessibility (WCAG AA) – Quick Launch

- Keyboard: Tab to navigate; Enter/Space to activate; focus ring visible
- Screen readers: aria-labels on all actionable elements; live regions for loading/status
- Color contrast: tokens ensure >= 4.5:1 contrast for body text; icons adapt to themes
- Roles and semantics: proper labeling for inline panels; avoid ambiguous button semantics
- Responsiveness: touch targets >= 44x44 px on mobile; responsive stacking of pills
