# Wireframes (Mid fidelity) – Quick Launch: Claude & ChatGPT

- Horizontal bar near editor chrome with two pill buttons: Claude and ChatGPT, plus optional tooltip text
- Each pill includes provider icon, label, and a small helper label on hover/focus
- Inline fallback panel (collapsed by default) that expands on popup-block or error
- Preview area showing Prompt V2 generated from current prompt
- Copy button for Prompt V2, and a small status indicator per provider

Accessibility notes:
- All interactive elements have ARIA labels; focus order is logical; color contrast verified.
